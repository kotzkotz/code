javascript小计

  1. 获取ID为content的内容:
    var content = document.getElementById('content').innerHTML;innerHTML内部是允许添加html代码的，如：id.innerHTML = "你好！";
    var content = document.getElementById('content').value;
  2. 中文或者空格转码方式：
    var str = '我是汉字';
    var enc = escape(str);//转成“%u6211%u662F%u6C49%u5B57”
    var enc = unescape(enc);
  3. js获取到页面的内容都是字符串，如果需要数字应使用：parseInt(str)。
    “isNaN”判断是不是数字。
  4. eval() 函数可计算某个字符串，并执行其中的的javascript代码。
    eval('alert("执行了")');
  5. 在对象中：
    obj.per==obj['per']而不是obj.per==obj[per]
